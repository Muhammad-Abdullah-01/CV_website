# CV Website – Assignment #1

## 📄 Project Information
This project is a personal **CV (Resume) website** created using **only HTML and CSS** as part of **Assignment #1**.  
The goal of this assignment is to practice clean layout design, responsiveness, and proper structuring without using JavaScript frameworks.

---

## 🛠️ Technologies Used
- **HTML5**
- **CSS3**
- **CSS Flexbox**

---

## ✨ Features
- Responsive design for different screen sizes  
- Clean and professional CV layout  
- Dark mode (CSS-only implementation)  
- Download CV feature using a pre-made PDF  
- Flexbox-based structure  
- Beginner-friendly, framework-free code
